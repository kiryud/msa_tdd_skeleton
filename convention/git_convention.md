# commit message

```
[prefix]_[content] : [message]
```
- commit message가 잘못되었거나 마음에 들지 않는다면 삭제가 아니라 다시 작성하자
- message는 자유롭게 작성하자

## prefix

- 추가
```
Add_[content] : [message]
```

- 수정
```
Fix_[content] : [message]
```

- 삭제
```
Del_[content] : [message]
```

- 구조 변경, 리팩토링 등 애매한 경우
```
Mod_[content] : [message]
```

## content

- 문서
```
[prefix]_Doc : [message]
```

- 코드
```
[prefix]_Code : [message]
```

- 테스트 코드
```
[prefix]_Test : [message]
```
